Animagic Animation Studio
Sep 2019 – May 2020 Capstone C / C++

Developed as part of the Entrepreneurial Capstone course at the University of Calgary. AniMagic is an animation suite that reduces the time, cost, and learning curve that animators face daily. It incorporates modern 2D and 3D tools in one place, allowing animators to maintain 2D charm while integrating 3D backgrounds automatically.

  • 3D model importing with scale, rotate, and translate controls.
  • 3D transformational keyframes per model.
  • Customizable animation playback speed with linear interpolation.
  • 3D model painting with custom color tools.
  • Proprietary file format (.magi) for save/load.
  • GIF and PNG export support.
